字幕自动生成-autosub

autosub github地址

安装依赖ffmpeg

使用依赖Google服务

使用方法：

autosub “音视频文件” -S “原视频语言” -D “字幕语言” -K “Google API KEY” -F “字幕格式”

如果原视频语言跟字幕语言不一致，则会使用Google Translate进行翻译。

支持输出的字幕格式

raw
json
vtt
srt
